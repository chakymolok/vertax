            локально-first → видны только те, кто запускал fit-check
          

          

            VERTAX хранит коллекцию пользователей в IndexedDB на клиенте. На сервер попадает только
            хэш и снапшот треков, когда юзер открывает «Подойдёт ли пластинка». Поэтому полная
            аналитика (заходы в день, графики Метрики-style) пока не на сервере — её надо тащить из
            Яндекс.Метрики через API.
          

          

          

            🚧 Графики и per-user данные — следующий заход. Сейчас доступно: общее число
            collection-index snapshot'ов, число уникальных user_id, число AI-вердиктов в кэше.
