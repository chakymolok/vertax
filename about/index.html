What is Vertax

A digital crate for physical records

Vertax is a vinyl DJ tool for diggers, collectors and vinyl-only DJ selectors. It helps you break down records, keep a clean vinyl collection, search metadata, run a BPM finder and key finder, and turn your shelves into playable DJ set builder ideas.

vinyl collection / Discogs import / Camelot wheel / harmonic mixing / drum and bass vinyl / vinyl-only DJ

The problem

Vinyl is tactile. Metadata is not.

Great records often arrive with almost no DJ metadata. When the collection grows, preparing a harmonic mixing flow becomes slow and messy, especially for fast styles like drum and bass vinyl.

  • BPM usually is not printed on records.
  • Key usually is not printed either.
  • Collection management can become messy.
  • After a live set it is hard to remember what exactly was played.
  • Building harmonic vinyl sets takes time.

Features

Built for digging, sorting and playing

BPM and Key finder

Search by artist and track name to find tempo, musical key and Camelot wheel hints for better transitions.

Vinyl collection

Keep records, labels, catalog numbers, years, styles and tracklists in a focused vinyl collection workflow.

Discogs import

Use Discogs import to bring your records into Vertax and start building a cleaner digital crate.

DJ set builder

Build sets by BPM, Camelot key and flow so a pile of records turns into a compatible running order.

Vinyl-only sets

Useful when you want the discipline of vinyl-only DJ sets but still need searchable musical metadata.
